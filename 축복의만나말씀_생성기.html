✝ 축복의 만나 말씀 생성기

Gemini API로 개역개정 성경 본문을 선정하고, 배경 이미지와 축복 문구까지 자동으로 만들어 카드 이미지를 완성합니다. API 키는 저장되지 않고 이 브라우저 세션에서만 사용됩니다.

1Gemini API 키
API 키가 없다면 Google AI Studio(aistudio.google.com)에서 무료로 발급받을 수 있습니다. 키는 서버에 저장되지 않고 이 화면에서만 사용돼요.
2제목 문구 & 날짜
최종 제목은 "문구 (M/D)" 형태로 표시됩니다. 예: 축복의 만나 말씀 (6/23)
3성경 본문
범위 안에서 가장 은혜로운 짧은 구절(1~3절)을 AI가 개역개정 기준으로 골라줍니다.
4하단 축복 문구
5배경 이미지
6로고 (선택)
